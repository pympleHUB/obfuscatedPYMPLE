## What is pympleHUB?

pympleHUB is a Roblox script hub that provides game-specific automation scripts, delivered through a secure key-gated loader. Scripts are hosted publicly and loaded at runtime — no manual updates needed on your end.

---

## How to Use

**1. Get the current key**
Join the [Discord server](https://discord.gg/KTZcYZzQFp) and grab the active key from the key channel. Keys rotate every **6 hours**.

**2. Execute the script**
Paste the following into your Roblox executor and execute:
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/pympleHUB/obfuscatedPYMPLE/refs/heads/main/MainLoaderUI"))()
```

**3. Enter your key**
A key verification UI will appear. Enter the current key from Discord to gain access.

## Key System

- Keys are **free** and rotate every **6 hours** automatically
- The current key is always posted in the Discord key channel
- A notification is sent to Discord when the key changes
- If you are in-game when the key rotates, you will be prompted to re-execute with the new key
